# Family Road Trip Plan — March 11–22, 2026

A 12-day road trip from Tamarac, FL through Savannah, GA and the NC Mountains.

## Live Site

[projects.aebdigitalconsulting.com/trip-plan](https://projects.aebdigitalconsulting.com/trip-plan/)

## How It Works

The trip plan is a single-page HTML app generated by a Python script. Weather forecasts are fetched live from the Open-Meteo API using a GFS + ECMWF multi-model blend for accuracy.

### Generate the HTML

```bash
python src/create_html.py
```

This outputs `road-trip-plan.html` which is copied to `index.html` for deployment.

### Project Structure

```
index.html                 # Generated HTML (deployed via GitHub Pages)
src/
  create_html.py           # Main HTML generator (Python f-string template)
  create_trip_docs.py      # Trip document generator
  patch_html.py            # HTML patching utility
  trip-plan.md             # Detailed trip plan source content
  fixes/                   # Incremental fix scripts applied to create_html.py
    fix_weather_blend.py   # Multi-model weather blend
    fix_lodging_links.py   # Lodging booking links
    fix_menu_*.py          # Navigation menu fixes
    fix_tables.py          # Location table reconciliation
    ...
```

### Weather API

Forecasts blend two models from [Open-Meteo](https://open-meteo.com):
- **GFS** (NOAA) — good general accuracy
- **ECMWF** — European model, often better for precipitation

Temperatures are averaged, precipitation takes the max (conservative for trip planning).
